Nightcode

Download Version 0.1.1
What is it?
Nightcode is the only thing you need to create Clojure and Java projects for the command line, the desktop, the web, and Android -- besides JDK 6 or above. Whether you choose to make a Clojure or pure Java project, Nightcode will use its built-in copy of Leiningen to build and run your code.

Join the discussion on /r/Nightcode.
Built-in templates
After hitting New Project and giving it a name, you'll be presented with eight ready-made templates. You can make a console program, a LibGDX game, and an Android app in either Clojure or Java. The game projects will compile for both Android and desktop from the same codebase! Additionally, there is a template for making a Clojure desktop program using the seesaw library, and for making a web app in ClojureScript.
Keyboard friendly
Hold down the control key (or command on OS X), and you'll see shortcut hints appear over every applicable widget. Holding it while pressing up/down/enter will move your selection in the project tree and expand/collapse directories. Holding it while pressing left/right will move between recent files, much like tabs in other IDEs but without the UI clutter. Lastly, paredit mode can be toggled for Clojure and ClojureScript files via a button above the editor.
Clojure integration
The REPL pane on the bottom-left is a convenient scratchpad to try out Clojure commands. The build pane on the bottom-right gives you access to all the Leiningen commands to run, build, and test your project. Additionally, Clojure files have a 'Doc' button above the editor, which will provide completion suggestions for whatever you're typing thanks to the integrated compliment library.
ClojureScript integration
Due to the integrated lein-cljsbuild plugin, building ClojureScript projects is supported. Using the provided template, you can build a complete web app using Clojure on the server side and ClojureScript on the client side -- all inside the same project. You'll even find an 'Auto' button in the build pane, which runs lein-cljsbuild's auto rebuild feature so your ClojureScript files rebuild every time you save them.
Android integration
Due to the integrated lein-droid plugin, building Android projects is supported. The "Run" button builds and installs your app on your connected device. The "Build" button creates a release version of the app and lets you sign it. In addition, Android projects have a special "LogCat" item in the sidebar, which will show your device's or emulator's logs.
Java integration
Nightcode works with pure Java projects as well, including standard Android projects. You can start one with one of the built-in templates, or you can import one and it will prompt you to create a project.clj file. You'll need to edit it a little bit so the Leiningen knows how to build the project. For non-Android Java projects, you'll get to use the Reload button, which will let you automatically inject newly-saved changes into your running program via Java's HotSwap feature.
